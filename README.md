## Featured Project

### KORKMAZ e-Belge Yönetim Sistemi

A JavaFX and MySQL based desktop document management system prototype with:

- XML / UBL-like document upload
- PDF document archive support
- Metadata management
- Role-based access control
- Audit logs
- Reporting
- SHA-256 file integrity verification

## How to Open Locally

Open the folder and double-click `index.html`.

## GitHub Pages Deployment

1. Create a GitHub repository for the portfolio.
2. Upload the files in this folder.
3. Go to `Settings > Pages`.
4. Select `Deploy from a branch`.
5. Choose `main` and `/root`.
6. Save. The portfolio link will be created after GitHub Pages finishes deployment.

## Notes

- Update project links in `index.html` when repository URLs change.
- Keep the CV file under `assets/`.
- For a cleaner GitHub Pages URL, use a repository name such as `portfolio` or `osmannkorkmaz06.github.io`.
